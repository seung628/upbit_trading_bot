# 📱 텔레그램 명령어 가이드

## 🎯 개요

텔레그램으로 봇에 명령어를 보내서 실시간으로 상태를 확인하고 제어할 수 있습니다!

---

## 🔧 설정 방법

### config.json
```json
{
  "telegram": {
    "enabled": true,
    "bot_token": "1234567890:ABCdefGHI...",
    "chat_id": "123456789",
    "enable_commands": true  ← 이것만 추가!
  }
}
```

그게 전부입니다! 프로그램 실행하면 자동으로 명령어 수신을 시작합니다.

---

## 📋 사용 가능한 명령어

### 📊 정보 조회

#### /status (또는 /상태)
**현재 거래 상태 확인**

```
📱 입력: /status

🤖 응답:
📊 현재 상태

🔄 상태: ▶️ 실행 중

💰 자금 현황
초기: 1,000,000원
현재: 950,000원
투자중: 200,000원
사용가능: 100,000원

📈 수익
총 평가액: 1,050,000원
총 수익률: +5.0%

📊 거래 통계
총 거래: 25회
승률: 64.0%
```

---

#### /daily (또는 /일일)
**오늘의 거래 통계**

```
📱 입력: /daily

🤖 응답:
📅 일일 통계

날짜: 2026-02-08

📊 거래: 15회
✅ 승: 10회
❌ 패: 5회
📈 승률: 66.7%

💰 총 손익: +12,500원

🏆 최고: BTC +3,450원
📉 최악: XRP -1,520원
```

---

#### /positions (또는 /포지션)
**현재 보유 중인 코인**

```
📱 입력: /positions

🤖 응답:
📍 보유 포지션

BTC
매수: 60,000,000원
현재: 61,500,000원
💰 수익: +2.5% (+2,475원)

ETH
매수: 3,000,000원
현재: 2,950,000원
📉 수익: -1.7% (-1,667원)
```

---

#### /balance (또는 /잔고)
**현재 잔고**

```
📱 입력: /balance

🤖 응답:
💰 잔고

원화: 850,000원
투자중: 200,000원
총 평가액: 1,050,000원
```

### 🎮 제어 명령어

#### /pause (또는 /정지)
**거래 일시 정지**

```
📱 입력: /pause

🤖 응답:
⏸️ 거래를 일시 정지했습니다.
/resume 으로 재개할 수 있습니다.
```

**효과:**
- 새로운 매수 중지
- 기존 포지션은 유지
- 매도는 계속 작동

**용도:**
- 시장이 불안정할 때
- 잠시 거래를 멈추고 싶을 때
- 급하게 매수를 막고 싶을 때

---

#### /resume (또는 /재개)
**거래 재개**

```
📱 입력: /resume

🤖 응답:
▶️ 거래를 재개했습니다.
```

**효과:**
- 거래 다시 시작
- 쿨다운도 해제

---

#### /help (또는 /도움말)
**명령어 도움말**

```
📱 입력: /help

🤖 응답:
📱 사용 가능한 명령어

📊 정보 조회
/status - 현재 상태
/daily - 일일 통계
/positions - 보유 포지션
/balance - 잔고 확인

🎮 제어
/pause - 일시 정지
/resume - 거래 재개

❓ /help - 이 도움말
```

## 💡 꿀팁

### 1. 빠른 체크
```
외출 중 빠르게 확인:
/status → 전체 상황
/positions → 수익 확인
```

### 2. 위험 회피
```
뉴스 보고 위험 감지:
/pause → 즉시 정지
나중에 /resume
```

### 3. 일일 루틴
```
아침: /resume (재개)
점심: /status (확인)
저녁: /daily (통계)
밤: /pause (정지)
```

## 🔒 보안 주의

### ✅ 안전
- 본인의 Chat ID에서만 작동
- 다른 사람이 명령어를 보내도 무시됨

### ⚠️ 주의
- 봇 토큰 노출 주의
- 공개 그룹에서 사용 금지
- 개인 채팅에서만 사용

## 🐛 문제 해결

### Q1. 명령어가 작동 안 해요
**A1:** enable_commands 확인
```json
{
  "telegram": {
    "enable_commands": true  ← 체크!
  }
}
```

**A2:** 프로그램 재시작
```bash
> stop
> exit
python main.py
> start
```

**A3:** 봇 Token/Chat ID 확인
```
TELEGRAM_SETUP.md 참고
```

---

### Q2. 응답이 느려요
**A:** 정상입니다!
- 텔레그램 API는 최대 30초 폴링
- 보통 1-5초 내 응답

---

### Q3. 명령어 오타
```
📱 /statsu
```
