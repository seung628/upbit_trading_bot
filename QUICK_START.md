# ⚡ 빠른 시작 가이드

## 5분 안에 시작하기

### 1️⃣ 설치 (1분)
```bash
pip install -r requirements.txt
```

### 2️⃣ API 키 설정 (2분)

1. 업비트 로그인
2. Open API 관리 → API 키 발급
3. `config.json` 파일 열기
4. 키 입력:
```json
{
  "api": {
    "access_key": "여기에_입력",
    "secret_key": "여기에_입력"
  }
}
```

### 3️⃣ 실행 (1분)
```bash
python main.py
```

### 4️⃣ 시작!
```
💻 명령어 입력 > start
```

---

## 핵심 설정 3가지

### 1. 투자 한도 설정
```json
{
  "trading": {
    "max_total_investment": 300000  // 최대 30만원
  }
}
```

### 2. 신호 강도 조절
```json
{
  "indicators": {
    "min_signal_score": 7  // 7점 이상만 거래
  }
}
```

### 3. 거래 시간 설정
```json
{
  "trading_hours": {
    "enabled": true,
    "sessions": [
      {"start": 9, "end": 11},   // 오전 9-11시
      {"start": 21, "end": 23}   // 밤 9-11시
    ]
  }
}
```

---

## 주요 명령어

```
start   - 거래 시작
stop    - 거래 정지 (포지션 청산)
status  - 현재 상태 확인
daily   - 오늘의 통계
exit    - 종료
```

## 위험 관리 필수 3가지

1. **손절 절대 지키기**
   - ATR 손절 신뢰
   
2. **투자 한도 넘지 않기**
   - max_total_investment 엄수
   
3. **욕심 부리지 않기**
   - 월 10-15%면 충분

---

## 문제 발생 시

### 거래가 안 돼요
→ `min_signal_score` 낮추기 (7 → 6)

### 손절이 너무 많아요
→ `atr_stop_loss_multiplier` 높이기 (1.5 → 2.0)

### 수익이 너무 적어요
→ 2주 이상 기다리기 (복리 효과)
